# Privacy Policy

## Data Processing

Text Fitter processes all text **entirely locally** within the Dify plugin runtime. No text data is transmitted to any external server, API, or third-party service.

## Data Collection

This plugin does **not** collect, store, or transmit:
- Input text content
- Processed/output text
- Usage statistics or telemetry
- User credentials or personal information

## Data Storage

All processing occurs in memory during tool execution. No data is persisted to disk or any external storage by this plugin.

## Third-Party Services

This plugin does not communicate with any third-party services. It uses only the Python standard library for its NLP algorithms (no external NLP APIs).

## Data Retention

No user data is retained after the tool execution completes. The plugin is stateless.

## Changes to This Policy

Any changes to this privacy policy will be reflected in updated versions of this plugin published to the Dify Marketplace.

## Contact

For privacy-related inquiries, please file an issue on the plugin's GitHub repository.
